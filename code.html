The Code

                    
function getValues() {
    let startValue = document.getElementById('startValue').value; 
    let endValue = document.getElementById('endValue').value; 

    let startNumber = parseInt(startValue); 
    let endNumber = parseInt(endValue); 

    let numberArray = generateNumbers(startNumber, endNumber);
    displayNumbers(numberArray);
}

function generateNumbers(start, end) {
    let basketOfNumbers = [];
  
    for (let number = start; number <= end; number = number + 1) {
        basketOfNumbers.push(number);       
    }

    return basketOfNumbers; 
}

function displayNumbers(numbers) { 

    let results = '';

    for (let index = 0; index < numbers.length; index++) {   
      
        let currentNumber = numbers[index];

        if (currentNumber % 2 == 0) {
            results += `<tr><td class="evenNumber"> ${currentNumber} </td></tr>`;
        } else {
            results =
              results + `<tr><td> ${currentNumber} </td></tr>`;
        }
    }    
    let tableBody = document.getElementById('results');
    tableBody.innerHTML = results;
}

This code is structured with three functions. Below you can get a better understanding of how this code works.

getValues( )

This function retrieves the values of the startValue and endValue elements from the HTML document. It then converts these values to integers using parseInt( ) and stores them in the variables startNumber and endNumber. Next, it calls the generateNumbers(startNumber, endNumber) function and passes the start and end numbers as arguments. Finally, it calls the displayNumbers(numberArray) function and passes the generated numbers as an argument.

generateNumbers(start, end)

This function takes two parameters, start and end, representing the range of numbers to generate. It initializes an empty array called basketOfNumbers. Then, it uses a for loop to iterate from the start number to the end number (inclusive). In each iteration, it adds the current number to the basketOfNumbers array using the push() method. After iterating through the range, it returns the array containing the generated numbers.

displayNumbers(numbers)

This function takes an array of numbers as a parameter. It initializes an empty string called results to store the HTML content. It then uses a for loop to iterate through each number in the numbers array. Inside the loop, it checks if the current number is even by using the modulo operator %. If the number is even, it appends a table row with the class "evenNumber" and the current number inside a table cell to the results string. If the number is odd, it appends a table row with only the current number inside a table cell to the results string. After the loop, it retrieves the results element from the HTML document and sets its innerHTML to the results string, which displays the generated numbers in an HTML table format.
